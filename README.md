# bilibili-lofi-girliaudio-player
A lightweight Windows tool to extract and play real-time audio from Bilibili Lofi Girl's live room, packaged as a standalone EXE , supporting silent running and automatic log recording./B 站 Lofi Girl 直播间音频提取播放工具，Windows 独立 EXE 版，支持带窗口 / 静默双模式，自动拉流转码、日志记录，适配 PotPlayer 播放
